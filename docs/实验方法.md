# 实验方法：GRAG增强的智能体记忆系统评估

## 1. 实验目标

验证GRAG（Graph Retrieval-Augmented Generation）增强的图结构记忆相比传统方法（FIFO、纯向量检索）在对话记忆检索中的优势。

### 核心假设
1. **图结构可以捕捉记忆间的多维关联**：语义、主题、时序三个维度
2. **PPR图传播可以增强跨主题检索**：通过图中路径发现间接相关的记忆
3. **PPR在对话记忆规模下可快速收敛**：迭代次数 < 50，延迟 < 100ms

## 2. 图构建方法

### 2.1 节点定义
每条对话记录为一个节点，包含：
- `role`: user / assistant
- `content`: 对话文本内容
- `topic`: 主题标签
- `timestamp`: Unix时间戳

### 2.2 边定义

**语义相似边** (cosine > 0.60)：
```
sim(v_i, v_j) = (e_i · e_j) / (||e_i|| * ||e_j||) > 0.60
```
每个节点最多保留 Top-8 语义边。

**同主题边** (same topic + sim > 0.40)：
```
topic_i == topic_j AND sim(v_i, v_j) > 0.40
```

**时序相邻边**（同主题相邻轮次）：
```
topic_i == topic_{i-1}
```

### 2.3 图传播

Personalized PageRank 迭代：
```
s(t+1) = α·pv + (1-α)·A·s(t)
```
- α = 0.85（重启概率）
- pv = 查询与各节点的余弦相似度（Personalization Vector）
- A = 列归一化邻接矩阵
- 终止条件：||Δ||₁ < 1e-6 或 max_iter=50

## 3. 分数融合

```
s_final = w * norm(vec_scores) + (1-w) * norm(ppr_scores)
```
- w = 0.6（向量权重）
- norm() = Min-Max归一化到[0,1]

## 4. 评估指标

| 指标 | 定义 | 计算方式 |
|------|------|----------|
| Topic Match Count | top-5中命中期望主题的数量 | sum(topic_i == expected) |
| Cross-Topic Coverage | 检索结果涉及多个主题的比例 | |topics_found| > 1 |
| PPR Convergence | 收敛所需迭代次数 | ||Δ||₁ < 1e-6 |
| Graph Density | 图稠密度 | 2|E|/(|V|(|V|-1)) |
| Avg Degree | 平均度数 | 2|E|/|V| |

## 5. 对比基准

- **FIFO基线**：仅保留最近K轮对话，不进行语义检索
- **纯向量检索**：仅使用cosine相似度排序，无图传播
- **GRAG（本文方法）**：向量相似度 + PPR图传播 + 分数融合
