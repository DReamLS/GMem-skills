# 评估过程：GRAG记忆系统实验复现

## 快速复现

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 运行全部实验
python -m experiments.run_experiments

# 3. 查看报告
cat results/experiment_report.json | python -m json.tool
```

## 实验流程

```
实验1: 构建记忆图
  ├── 加载5主题17轮对话 -> 34节点
  ├── 增量构建语义/主题/时序边
  └── 验证图结构完整性

实验2: 纯向量 vs GRAG 检索
  ├── 对5个查询执行纯向量检索
  ├── 对相同查询执行GRAG检索
  └── 对比top-5中期望主题的命中数

实验3: 跨主题检索
  ├── 选取5个跨主题查询
  ├── 执行GRAG检索
  └── 评估多主题覆盖率

实验4: 图结构分析
  ├── 计算度分布、密度、聚类系数
  └── 识别枢纽节点

实验5: PPR收敛性
  ├── 对不同查询运行PPR
  └── 记录收敛迭代次数

实验6: 边界情况
  ├── 空记忆库
  ├── 相似内容去重
  ├── 增量存储
  └── 图对称性
```

## 输出文件

```
results/
├── experiment_memories.json    # 实验记忆数据
├── experiment_report.json      # 完整实验报告
└── empty_test.json             # 边界测试临时数据
```

### 预期结果

| 指标 | 预期值 | 说明 |
|------|--------|------|
| 跨主题成功率 | > 60% | GRAG应超过纯向量 |
| PPR收敛 | < 15次迭代 | 对话规模图 |
| 检索延迟 | < 100ms | 满足实时需求 |
| 图密度 | 10-50% | 适中连通性 |

## 复现确认

执行 `python -m experiments.run_experiments` 后：
1. 控制台打印每个实验的详细结果
2. `results/experiment_report.json` 生成完整报告
3. 若所有实验均无异常退出，则复现成功

## 自定义实验

可通过修改 `run_experiments.py` 中的 `TEST_CONVERSATIONS`、
`VECTOR_VS_GRAG_QUERIES` 等变量来扩展实验数据。
