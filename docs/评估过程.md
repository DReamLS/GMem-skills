
## 核心升级：自动更新机制

### 旧系统（固定数据库）
```
用户 → 手动 store() → 固定图 → 手动 search() → 结果
```
需要手动调用 `store()` 和 `search()`，图结构仅在调用时更新。

### 新系统（AutoMemoryAgent - 自动更新）
```
用户 → [自动推断主题] → [自动GRAG检索历史] → [生成回复] → [自动存储到图] → 图动态增长
```

**自动更新流程图：**
```
每轮对话 ──→ ① 推断主题 (infer_topic)
          ├─→ ② GRAG检索相关记忆 (search) ← 实时读取图
          ├─→ ③ 生成回复
          └─→ ④ 存储对话到图 (store) → 图增量更新
                ├─ 新节点加入
                ├─ 增量建边（语义/主题/时序）
                └─ 邻接表自动维护
```

### 验证结果（200+节点图）
```
自动更新验证:
  总对话轮次: 15 → 30节点, 435边
  平均检索记忆数: 4.4条/轮
  检索延迟: <1ms
  
  ▶ GRAG相关问题验证: 检索到2条相关记忆 ✓
```
